Ajustes de instalación

El número de LEDs se guarda en el propio ESP32, que se reinicia al guardar (la app se desconecta unos segundos; vuelve a conectar).

Si una tira muestra los colores cambiados (p. ej. el rojo sale verde), prueba otro orden hasta que el rojo sea rojo:

Clave del Victron (VictronConnect → Info del producto → Instant readout → clave de cifrado). Vacío = desactivado.

Actualizar firmware por WiFi (OTA): el ESP abre su propia red LucesAC-OTA (clave luces-ac) con una página para subir el .bin. Sin cables ni desmontar nada.
